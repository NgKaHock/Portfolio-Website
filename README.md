# 🌐 Portfolio Website

## 👋 Overview

This is my personal portfolio website designed to showcase my software development projects, technical skills, and professional work. It serves as a centralized space where my projects, abilities, and technical progress are presented in a clear and structured way.

## 🎯 Purpose

The goal of this project is to build a professional online presence that clearly represents me as a developer. It helps others quickly understand my capabilities, the work I have done, and the types of solutions I can build. It also supports collaboration opportunities and project visibility.

## 🚀 Features

* 🧩 Project showcase section to display my work
* 👤 About section introducing my background and interests
* 🛠️ Skills section highlighting my technical stack
* 📱 Responsive design for both desktop and mobile devices

## 💻 Tech Stack
* HTML / CSS / JavaScript

* Home
